# F13 카카오톡 채널 — 소개문구/인사말 초안

실제 채널 개설(개인/사업자 인증, URL 연결)은 사용자가 [카카오톡 채널 관리자센터](https://center-pf.kakao.com/)에서 직접 진행. 아래는 그 화면에 붙여넣을 문구 초안.

## 채널 홈 소개문구 (프로필 설명, 최대 150자 내외)

```
로또 번호 생성기 + 가까운 판매점 찾기를 한 번에.
통계 기반 번호 조합, 예산별 추천 게임 수, 1등 배출 판매점 지도까지 무료로 이용하세요.
당첨을 보장하지 않는 재미 목적의 서비스입니다.
```

## 챗봇 인사말 (첫 대화 시 자동 발송)

```
안녕하세요! 로또 미니앱입니다 🎱

여기서 할 수 있는 것
- 통계 기반 번호 생성기 (전략별 조합)
- 예산에 맞춘 게임 수 추천
- 역대 최다출현 번호 TOP 6
- 내 주변 판매점 찾기 (1등 배출 이력 포함)

아래 버튼을 눌러 바로 시작하세요.
※ 번호 생성 결과는 통계적 재미 요소이며 당첨을 보장하지 않습니다.
```
